Quote Detail Screen

Displays a single quote with extended information including:

  • Author image and name
  • Full quote text
  • Biographical information about the author
  • Historical context for the quote
  • Related quotes in the same category

The screen uses a rich visual layout with the deep purple theme consistent with the rest of the application, and provides navigation back to the quotes list.
